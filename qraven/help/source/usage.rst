Usage
=====

.. _Usage:


Open QRaven
------------
Click on the QRaven icon in your toolbar image or go in the "Plugins" menu, select the QRaven option and click on "Generate Raven input files"
You will have four main tabs

 * Raven RVI
 * BasinMaker RVH
 * GridWeights
 * Run Model
